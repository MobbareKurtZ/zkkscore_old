# ztek

## Project setup
```
npm install
```

### Run frontend in electron
```
npm run electron:serve
```
### Run frontend
```
npm run serve
```

### Run server
```
cd src/functions && export FLASK_APP=server && flask run --host=0.0.0.0
```
